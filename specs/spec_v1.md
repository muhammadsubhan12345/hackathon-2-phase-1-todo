# Todo App Specification v1

## Objective
Create a command-line todo application in Python.

## Data Model
A Todo contains:
- id: integer
- title: string
- description: string
- completed: boolean

## Features
1. Add a todo
2. View all todos
3. Update a todo
4. Delete a todo
5. Mark todo as complete/incomplete

## Constraints
- In-memory storage only
- Python console application
